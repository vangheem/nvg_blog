--- layout: post title: "p4a on Plone 3.1 buildout" date: 2008-09-13 10:20:00 categories: description: Getting p4a.ploneaudio, p4a.plonevideo is a little difficult on Plone 3.1. Here is the buildout that shows how to do it. ---


# Add additional eggs here
# elementtree is required by Plone
eggs =
    elementtree
    p4a.plonevideo>=1.1,<=1.1.9999
    p4a.plonevideoembed>=1.1,<=1.1.9999
    p4a.ploneaudio
    p4a.plonecalendar
    p4a.audio
    hachoir-core
    hachoir-parser
    hachoir-metadata
    Products.basesyndication
    Products.fatsyndication
   
# Reference any eggs you are developing here, one per line
# e.g.: develop = src/my.package
develop =
    src/p4a.ploneaudio
    src/p4a.audio
   
# If you want to register ZCML slugs for any packages, list them here.
# e.g. zcml = my.package my.other.package
zcml =
    p4a.z2utils
    p4a.subtyper
    p4a.common
    p4a.fileimage
    p4a.audio
    p4a.ploneaudio
    p4a.video
    p4a.videoembed
    p4a.plonevideo
    p4a.plonevideoembed
    p4a.plonecalendar-meta
    p4a.plonecalendar
    p4a.ploneevent
